# Docit Sanbox Demo New

# What's Docit?

san docit 命令是放在packages/san-cli-docit中实现的，它是一个 Commander 插件，如果要编写自己的 Commander 插件可以参考它的代码。

<template>
    <div id="hello-codebox">
        <hello />
        <h2>{{text}}</h2>
    </div>
</template>
<style lang="less">
    @red: red;
    #hello-codebox {
        h1 {
            font-size: 18px;
        }
        h2 {
            font-size: 16px;
            color: @red;
        }
    }
</style>
<script>
    import Hello from './component.js';
    export default {
        initData() {
            return {
                text: 'Red, Less enabled!'
            };
        },
        components: {
            hello: Hello
        }
    };
</script>
